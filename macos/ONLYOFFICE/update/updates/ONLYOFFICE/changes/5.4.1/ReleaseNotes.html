ONLYOFFICE 5.4.1 - September 26, 2019

New features

All editors

  • Added more options for Paragraph spacing;
  • Added the possibility to set paragraph Outline level;
  • Added the possibility to get current color scheme;
  • Updated the Document Info page;
  • Added better templates for tables and increased their number;
  • Added the possibility to select languages using keyboard (bug #24317);
  • Added the possibility to show a shadow for images and shapes;
  • Added the possibility to use mentions in comments;
  • Added the possibility to show multi-gradient fill in shapes (bug #40719);
  • Added the onRequestSaveAs API event to save files to a storage;
  • Added the onRequestInsertImage event and insertImage method to insert images from a storage.

Document Editor

  • Added the possibility to create a new style with the next style being the same as previous one;
  • Added the possibility to add watermarks;
  • Added the possibility to print the selection;
  • Added the possibility to save DOCX using the mode compatible with older versions;
  • Added the tooltip for table styles;
  • Added the onRequestMailMergeRecipients event and the setMailMergeRecipients method for mail merge.

Spreadsheet Editor

  • Added the possibility to change look of chart elements;
  • Added the possibility to set headers and footers;
  • Added a new tab to work with formulas;
  • Added the possibility to group data;
  • Added the spellchecker;
  • Added the support of print areas;
  • Added the possibility to resize the autofilter window;
  • Added the possibility to show the number of cells in autofilter window;
  • Added the possibility to save 10 last used functions;
  • Set default value for regional settings (bug #41549);
  • Added support for asynchronous loading of formulas.

Presentation Editor

  • Added new themes for slides;
  • Added the support for the print selection option for slide;
  • Added the possibility to set headers and footers;
  • Added the possibility to insert the slide number;
  • Added the possibility to insert date and time;
  • Added the tooltip for table styles.

Fixes

All editors

  • Fixed a lot of bugs.
